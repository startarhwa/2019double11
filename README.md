### 使用说明
******
1. 下载并安装double11
2. 打开淘宝喵铺主页 
3. 运行double11
4. 脚本会检测无障碍模式，若未开启会自动跳转至设置页面，请手动开启后返回至double11运行页面以免脚本运行失败
5. 音量+键可以终止脚本
